## ğŸ”§ Setup

### 1. Get Your Lingo.dev API Key

1. Sign up at [lingo.dev](https://lingo.dev)
2. Get your API key from the dashboard
3. Add it as `LINGODOTDEV_API_KEY` environment variable or pass via input

### 2. Run Locally


